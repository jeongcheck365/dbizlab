## 리라이팅 규칙 (중요)

### 건드리지 않는 영역
- 헤더 (HEADER)
- 입력폼 (FORM SECTION)
- 푸터 (FOOTER)
- 스크립트
- CSS 클래스명

### 리라이팅 = 문장 새로 작성
- 브랜드명만 바꾸는 게 아님
- 의도/맥락/글자수 유지하면서 표현 완전히 변경

### BAS 금지 표현
성공사례, 성공금액, 성공률, 대출알선, 대출소개, 서류대행, 기업평가, 기업분석, 기업진단, 정책자금, 정부정책자금, 승인, 승인률
