## 리라이팅 규칙

1. **의미 동일** - 핵심 메시지는 유지
2. **표현 변경** - 문장 구조, 어순, 동의어 활용
3. **글자수 유지** - ±10% 이내
4. **자연스러운 한국어** - 어색하지 않게
5. **CSS 클래스명 유지** - keai-* 클래스는 코드이므로 변경 X

### 리라이팅 예시
```
원본: "기업의 성장을 위한 최적의 자금 솔루션을 제공합니다"
변경: "비즈니스 확장에 필요한 맞춤형 자금 지원 방안을 안내해 드립니다"
```
