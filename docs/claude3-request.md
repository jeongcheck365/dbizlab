## 상세 규칙

### 리라이팅 정의
**리라이팅 = 의도, 맥락, 글자수를 유지하면서 문장 자체를 새로 작성**

| 원칙 | 설명 |
|------|------|
| 의미 동일 | 핵심 메시지 100% 유지 |
| 표현 변경 | 문장구조, 어순, 동의어 전면 활용 |
| 글자수 유지 | ±10% 이내 |
| 자연스러운 한국어 | 번역체 X |

### 건드리지 않는 영역
- ❌ 헤더 (네비게이션, 로고)
- ❌ 입력폼 (무료심사 신청)
- ❌ 푸터 (연락처, 사업자정보)
- ❌ 스크립트/JS
- ❌ CSS 클래스명

### 리라이팅 대상
- ✅ 히어로 섹션 (제목, 설명)
- ✅ 서비스/프로세스 설명 섹션
- ✅ 특징/장점 섹션
- ✅ CTA 섹션 (폼 제외한 텍스트)
